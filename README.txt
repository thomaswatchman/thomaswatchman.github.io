  ## contact.js
  - Purpose: JavaScript code for handling form submission and displaying messages based on form validation.
  - Dependent Code Structures: N/A

  ### Functions
  1. `showMessage(message, type)`
      - Purpose: Displays a message on the message panel.
      - Input:
          - `message`: The message to be displayed.
          - `type`: The type of message, either "error" or "success".
      - Example: `showMessage("All fields are required", "error")`

  ### Event Listeners
  1. `form.addEventListener("submit", async function (event))`
      - Purpose: Listens for form submission event and performs form validation.
      - Input:
          - `event`: The event object.
